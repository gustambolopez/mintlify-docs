## Light Mode Palette

**Light mode** emphasizes clarity, brightness, and high contrast. Colors are defined using `oklch()` for perceptual consistency.

| Category               | Token                    | Color                       | Description                                     |
| ---------------------- | ------------------------ | --------------------------- | ----------------------------------------------- |
| Primary                | `--color-primary`        | `oklch(0.205 0 0)`          | Used for main actions, buttons, and highlights. |
| Primary Foreground     | `--color-primary-fg`     | `oklch(0.985 0 0)`          | Text or icons displayed on primary backgrounds. |
| Secondary              | `--color-secondary`      | `oklch(0.97 0 0)`           | Used for secondary actions or subtle emphasis.  |
| Secondary Foreground   | `--color-secondary-fg`   | `oklch(0.205 0 0)`          | Text on secondary backgrounds.                  |
| Accent                 | `--color-accent`         | `oklch(0.97 0 0)`           | Decorative elements or subtle highlights.       |
| Accent Foreground      | `--color-accent-fg`      | `oklch(0.205 0 0)`          | Text or icons on accent backgrounds.            |
| Background             | `--color-background`     | `oklch(1 0 0)`              | Page background.                                |
| Foreground             | `--color-foreground`     | `oklch(0.145 0 0)`          | Default text color.                             |
| Card                   | `--color-card`           | `oklch(1 0 0)`              | Card or panel backgrounds.                      |
| Card Foreground        | `--color-card-fg`        | `oklch(0.145 0 0)`          | Text inside cards.                              |
| Popover                | `--color-popover`        | `oklch(1 0 0)`              | Popover or tooltip background.                  |
| Popover Foreground     | `--color-popover-fg`     | `oklch(0.145 0 0)`          | Text inside popovers.                           |
| Muted                  | `--color-muted`          | `oklch(0.97 0 0)`           | Less prominent text or UI elements.             |
| Muted Foreground       | `--color-muted-fg`       | `oklch(0.556 0 0)`          | Text displayed on muted backgrounds.            |
| Destructive            | `--color-destructive`    | `oklch(0.577 0.245 27.325)` | Warnings, errors, or destructive actions.       |
| Destructive Foreground | `--color-destructive-fg` | `oklch(0.985 0 0)`          | Text on destructive backgrounds.                |
| Border                 | `--color-border`         | `oklch(0.922 0 0)`          | Borders for inputs, cards, and dividers.        |
| Input                  | `--color-input`          | `oklch(0.922 0 0)`          | Form input backgrounds.                         |
| Ring                   | `--color-ring`           | `oklch(0.708 0 0)`          | Focus indicators.                               |

## Dark Mode Palette

**Dark mode** focuses on contrast, readability, and depth for low-light environments. The semantic tokens remain consistent but colors are adjusted for visibility.

| Category             | Token                  | Color              | Description                  |
| -------------------- | ---------------------- | ------------------ | ---------------------------- |
| Primary              | `--color-primary`      | `oklch(0.922 0 0)` | Main actions and highlights. |
| Primary Foreground   | `--color-primary-fg`   | `#0e0e0e`          | Text on primary backgrounds. |
| Secondary            | `--color-secondary`    | `oklch(0.269 0 0)` | Secondary actions.           |
| Secondary Foreground | `--color-secondary-fg` | `oklch(0.985 0 0)` | Text for secondary items.    |
| Accent               | `--color-accent`       | `#262626`          | Decorative accents.          |
| Accent Foreground    | `--color-accent-fg`    | `oklch(0.985 0 0)` | Text on accent backgrounds.  |
| Background           | `--color-background`   | `oklch(0.145 0 0)` | Page background.             |
| Foreground           | `--color-foreground`   | `oklch(0.985 0 0)` | Default text.                |

_(Other tokens follow same semantic structure as light mode, adjusted for dark contrast.)_

## Usage

Include the palette in your css or you can get it from [Dark mode](https://cdn.jsdelivr.net/gh/outbrowsed/design-system@main/cdn/themes/dark.css) [Light mode](https://cdn.jsdelivr.net/gh/outbrowsed/design-system@main/cdn/themes/dark.css):

```css example.css
button.primary {
  background-color: var(--color-primary);
  color: var(--color-primary-fg);
}

.card {
  background-color: var(--color-card);
  color: var(--color-card-fg);
}

body {
  background-color: var(--color-background);
  color: var(--color-foreground);
  border-radius: var(--radius);
}
```
