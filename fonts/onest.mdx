**Onest** is a typeface developed by designers Dmitri Voloshin and Andrey Kudryavtsev from Moldova, commissioned to enhance public‑communication clarity and neutrality, it is very similar to [Geist](./geist).

Built as a hybrid of geometric and humanistic grotesques, Onest is optimized for both digital interfaces and longer reading contexts.

It supports both Latin and Cyrillic scripts, includes multiple weights (Thin → Black), and is licensed under the SIL Open Font License (OFL 1.1).

**Using Onest on the Web:**

You can get it from google fonts:

```html onest.html
<link href="https://fonts.googleapis.com/css2?family=Onest:wght@100..900&display=swap" rel="stylesheet">
```

Then apply use with:

```css onest.css
body {
    font‑family: 'Onest', sans‑serif;
}
```
